## 🔍 Frequently Asked Questions

What if the installer does not start?
Ensure you have an active internet connection if you are using a web-based installer. Check that your Windows account has administrator rights.

Can I install this on a Mac?
No, this version is specifically for Windows 11 and Windows 10.

Does the software need an update?
The software checks for updates automatically when it starts. If you want to check manually, look for an "Update" button inside the program settings menu.

Is there a trial period?
The version downloaded from the release link includes full features. You may need to activate it with a key provided by SoftMaker if you intend to use it for long-term production work.

Do I need to uninstall my old office software?
It is not strictly necessary, but running two office suites at the same time can sometimes confuse your computer when opening default files. You may prefer to remove your previous software to avoid issues with file association.

Where do my documents save by default?
The software saves files to the "Documents" folder on your computer unless you choose a different location. You can change this path in the program preferences.

How do I remove the software?
Open the Windows Settings menu, go to "Apps," select "Installed apps," find SoftMaker Office in the list, and select "Uninstall." Follow the prompts to remove the software from your system.
